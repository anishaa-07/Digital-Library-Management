# Digital-Library-Management
Web-based digital library system with admin control and user management.

A web-based Digital Library Management System designed to automate and simplify library operations. The system manages book records, user accounts, issuing and returning books, fine calculation, and report generation through a centralized database. It provides separate access for administrators and users, ensuring secure and efficient library management.
